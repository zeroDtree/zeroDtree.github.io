## 2. 正文

人的存在 $\iff$ 一个物理实体（身），在环境中持有与交换资源（物/权），处于社会网络中（位），用意识解释一切（故事），并做出行动（能）。

安全感：对存在的连续性不受威胁的确信。

## 3. 安全的范畴

- 实体性安全

  ```
  核心对象：作为生物有机体的“身”。

  连续性的体现：肉体生命与生理功能的完整、健康、持续运行。
  ```

- 持有性安全

  ```
  核心对象：作为资源调配者的“物”与“权”。

  连续性的体现：对维持与拓展生存所需的物质资源、社会资源及对其的支配权的稳定占有与控制。
  ```

- 联结性安全

  ```
  核心对象：作为关系节点的“位”。

  连续性的体现：在社会关系网络与意义系统中，拥有一个稳定、可预期且被认可的位置。
  ```

- 叙事性安全

  ```
  核心对象：作为意义生成者的“故事”。

  连续性的体现：对个人经历与外部世界有一套连贯、自洽、有价值的解释模型，使得经验可被理解，未来可被期待。
  ```

- 能动性安全

  ```
  核心对象：作为行动发起者的“能”。

  连续性的体现：意志的自由、选择的可能及行动的有效性得以持续。
  ```

## 4. 策略

- 时间维度
  ```
  - 进取：向未来扩张
  - 保守：退回过去防守
  ```
- 空间维度
  ```
  - 连接：靠近他者以求共生
  - 解离：远离他者以求自存
  ```
