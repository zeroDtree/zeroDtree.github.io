## 前提

人总是很反感别人给自己讲道理

1. 权力位阶的防御（被降维打击的挫败感）
	```
	当一个人给另一个人讲道理时，
	潜台词往往是：“我比你看得透，我比你更优越。”
	讲道理建立了一个瞬间的“高低位”。
	听者会被迫进入“学生”或“被修正者”的角色。
	这种地位的不平等会触发本能的心理防御。
	```
2. “认知失调”与自我连贯性
	```
	每个人都有一套维持自我认知的“防火墙”。
	如果别人的道理直接否定了你的行为，
	你的大脑会产生极大的焦虑（认知失调）。
	承认对方的道理，往往意味着承认“我以前错了”或“我以前很蠢”。
	为了保护自尊，大脑会自动化地寻找对方逻辑里的漏洞进行反击。
	```
3. 被剥夺的“发现权”
	```
	人类对知识的获取有一种近乎狩猎的本能渴望。
	直接喂到嘴里的标准答案是“死的”，
	它剥夺了你思考、联想、挣扎、最后恍然大悟的快感。
	就像玩解谜游戏，如果别人直接告诉你密码，你会觉得索然无味，甚至厌烦那个剧透的人。
	```

## 结论

写小说的人都是狡猾的狐狸

狡猾在于不直接讲给你道理，而用故事诱导你，让你觉得仿佛是自己的感悟。

ta给你留白，当你拼凑出那个真相时，你会深信那是**我的发现**，而非**ta的教导**。

而人类很少反驳自己得出的结论。

ta让你在安全的虚构世界里，“替”主角犯错，并承受后果。

因为死掉的是“主角”而不是“我”，所以读者的自尊心不会受损。在“替”主角反思的过程中，读者悄悄完成了自我认知的改变，却不需要经历承认失败的痛苦。
