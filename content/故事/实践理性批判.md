## 2. 命令

### 2.1. 假言命令

假言命令是一种以某个目的为前提的命令，形式是：

```
如果你想某个目的，那么你就应当采取某种手段。
```

它的约束力取决于你是否真的想要那个目的。

例子

```
如果你想通过考试，你就应该复习。
如果你想发财，你就应该努力工作。
如果你想被尊重，你就应该守信。
```

### 2.2. 定言命令

定言命令是一种不依赖任何目的、不以任何条件为前提的命令，形式是：

```
你应当这样做。
```

不管你想要什么，它都对你有效。不是因为你想要A，而是因为这是你作为理性存在者本身应当做的事。

例子

```
你应当尊重他人。
你不应当把别人仅仅当作工具。
```

## 3. 道德

道德是一种由理性存在者为自身立法的、普遍必然的、以定言命令形式表现出来的自律性法则体系，其本质在于意志的准则符合理性普遍化形式，并将理性存在者（人性）在任何时候都尊为目的本身。

纯粹实践理性是指不依赖于任何经验性动机（如欲望、情感、利益），仅凭其自身就能决定意志、产生道德法则并驱动道德行动的理性能力。

最高的道德(元道德)不是一条具体规则，而是道德立法的“形式原则”本身，即：

```
“你要只按照你同时能够愿意它成为一条普遍法则的那个准则去行动。”
（定言命令的“普遍法则公式”）
```

以及它的另一面：

```
“你要如此行动，即无论是你人格中的人性，还是其他任何一个人的人格中的人性，
你在任何时候都同时当作目的，而绝不仅仅当作手段来使用。”
（定言命令的“人性公式”）
```

## 4. 自由

自由无法在理论上被“证明”，但能在实践上被“确证”。

论证核心：道德与自由的互证

```
无可否认的事实：你内心能直接感知到 无条件的道德命令（如“尊重他人”）。

先验分析：这种无条件的命令若要可能，唯一前提是你的意志是自由的——即能独立于欲望因果，仅凭理性自我决定。
```

结论：

```
自由是道德的存在理由（没有自由，道德不可能）。
道德是自由的认识理由（通过道德意识，你认识到自己是自由的）。
```

你无法证明自由存在，但每当你做出一个道德选择时，你都在活生生地践行自由。 自由是道德行动的前提与显现。
