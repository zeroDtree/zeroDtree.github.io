## 幂运算

可以在有序(存在一个正元集)整环(非零、无零因子、交换幺环)$R$上递归定义幂运算$x^n$，其中$x \in R$，$n \in \mathbb{N}$

$x^0=1$

$x^n=x^{n-1}x$

#### 性质

1. $x^n x^m = x^{n+m}$
2. $(x^n)^m = x^{nm}$
3. $(x y)^n = x^n y^n$
4. 若$n > 0$，则$x^n = 0$当且仅当$x = 0$
5. 若$n>0 \wedge x \gt y \gt 0$，则$x^n \gt y^n \gt 0$。
6. $|x^n| = |x|^n$

若$R$也是域，则可以定义整数幂

$x^{-n}=(x^{-1})^n$ for $x\not=0,n\in {\mathbb{N}}$

1. $x^n x^m = x^{n+m}$

   - 若$m,n\geq 0$，则$x^n x^m = x^{n+m}$自然成立
   - 若$m,n\leq 0$，则$x^n x^m = (x^{-1})^{-n} (x^{-1})^{-m} = x^{-n-m} = x^{n+m}$
   - 若$m\geq 0,n\lt 0$，对$m$使用数学归纳法

2. $(x^{-1})^n = x^{-n}$
3. $(x^n)^{-1} = (x^{-1})^n$
4. $(x^n)^m = x^{nm}$
5. $(x y)^n = x^n y^n$
6. If $x \gt y > 0$, then $x^n \gt y^n > 0$ if $n$ is positive, and $0 < x^n \lt y^n$ if $n$ is negative.
7. If $x, y > 0$, $n \not= 0$, and $x^n = y^n$, then $x = y$.
8. $|x^{-1}| = |x|^{-1}$
9. $|x^n| = |x|^n$
