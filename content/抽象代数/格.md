## 定义

**格（Lattice）**：如果偏序集 $E$ 中的任意两个元素 $x, y$，它们的上确界（最小上界）和下确界（最大下界）都存在，那么 $E$ 就被称为一个格。

我们定义两个运算：

- 连接（Join / 极大值）：$\max(x, y) = x \vee y$ （即 $\{x, y\}$ 的最小上界）
- 交（Meet / 极小值）：$\min(x, y) = x \wedge y$ （即 $\{x, y\}$ 的最大下界）

向量格（Vector Lattice / Riesz Space）如果一个实向量空间 $E$ 同时是一个格，并且它的偏序结构与线性结构相容，即：

- 加法平移不变性：若 $x \le y$，则对任意 $z \in E$，有 $x + z \le y + z$。
- 正标量乘法不变性：若 $x \le y$，则对任意实数 $c \ge 0$，有 $cx \le cy$。

那么 $E$ 称为**向量格**。

在向量格中，我们可以借助 $0$ 元素正式定义绝对值：$$|x| = x \vee (-x)$$

**Banach 格**（Banach Lattice）：一个赋范线性空间 $E$ 如果同时是向量格，并且范数与模长（绝对值）满足单调性：$$\text{若 } |x| \le |y|, \quad \text{则 } \|x\| \le \|y\|$$若 $E$ 在该范数下完备，则称 $E$ 为 Banach 格。
