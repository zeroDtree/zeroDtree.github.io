## 功能性插件

### Python

| 插件名          | 功能                                                                        |
| --------------- | --------------------------------------------------------------------------- |
| Python          | 核心基础设施：环境管理、解释器选择、单元测试集成（ty 与 Ruff 依赖此插件）   |
| ty              | 语言服务 (LSP)：类型检查、代码补全、定义跳转，替代 Pylance                  |
| Ruff            | 静态分析：极速 Lint 检查、代码格式化、导入排序，替代 Black / isort / Flake8 |
| Python Debugger | 调试引擎：断点调试、变量监控、堆栈追踪                                      |
| autoDocstring   | 自动生成 Docstring 模板                                                     |
| Jupyter         | Notebook 支持                                                               |

### 前端 / 全栈

| 插件名                            | 功能                 |
| --------------------------------- | -------------------- |
| Vue - Official                    | Vue 语法支持         |
| JavaScript and TypeScript Nightly | JS / TS              |
| Spring Boot Extension Pack        | Spring Boot 开发套件 |

| 插件名                        | 功能                            |
| ----------------------------- | ------------------------------- |
| Markdown All in One           | Markdown 语法支持、快捷键、预览 |
| Markdown Preview Enhanced     | Markdown 预览增强               |
| Markdown TOC & Chapter Number | 章节编号与目录自动生成          |
| LaTeX Workshop                | LaTeX 语法支持、编译、预览      |
| LaTeX Formatter               | LaTeX 代码格式化                |
| Tinymist Typst                | Typst 语法支持                  |
| Paste Image                   | 在markdown里粘贴图片            |

### AI 插件

| 插件名                              | 功能                                                                                                                           |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Claude Code for VS Code             | 在vscode中使用claude code。[deepseek支持](https://github.com/deepseek-ai/awesome-deepseek-agent/blob/main/docs/claude_code.md) |
| OAI Compatible Provider for Copilot | 使 Copilot 支持 OpenAI 兼容 API（如 DeepSeek）                                                                                 |
| DeepSeek V4 for Copilot Chat        | 在 Copilot Chat 模型选择器中直接使用 DeepSeek V4 Pro / Flash，支持思考模式、视觉代理与工具调用，BYOK                           |
