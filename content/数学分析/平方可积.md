## 定义

在所有平方可积的复（或实）函数集合上，定义一个二元关系 “$\sim$”：$$f \sim g \iff f(x) = g(x) \quad \text{a.e.}$$易证该关系满足自反性、对称性与传递性，因而是严格的等价关系。通过该关系，可将原函数空间划分为无数个等价类（Equivalence Classes），记作 $[f]$：$$[f] = \{ g \mid g(x) = f(x) \ \text{a.e.} \}$$

严格数学意义上的 $L^2([a, b])$ 空间 并非函数的集合，而是所有平方可积函数等价类的商空间（Quotient Space）：$$L^2([a, b]) := \left\{ [f] \ \middle|\  \int_{a}^{b} |f(x)|^2 dx < +\infty \right\}$$

等价类上的线性运算与积分对于空间中的元素（等价类） $[f], [g]$ 以及标量 $\alpha \in \mathbb{C}$，其运算严格定义为：

- 加法：$[f] + [g] := [f + g]$
- 数乘：$\alpha [f] := [\alpha f]$
- 内积映射：$\langle [f], [g] \rangle := \int_{a}^{b} f(x)\overline{g(x)} dx$
- 良定性（Well-definedness）注记：测度论保证了若 $f_1 \sim f_2$ 且 $g_1 \sim g_2$，则 $\int_{a}^{b} f_1\overline{g_1} dx = \int_{a}^{b} f_2\overline{g_2} dx$。这意味着无论从等价类中任取哪个“代表元”进行积分，其结果完全相同且唯一确定。

为了书写简便，后续证明中我们将等价类 $[f]$ 直接简记为 $f$，但其本质依然代表整个等价类。

设标量域为复数域 $\mathbb{F} = \mathbb{C}$（实数域只需去除共轭符号，证明完全兼容）。

### 核心部分 I：向量空间公理（线性空间的封闭性）

要证明 $L^2([a, b])$ 是一个向量空间，必须证明它对线性组合保持封闭。设任意 $f, g \in L^2([a, b])$ 且 $\alpha, \beta \in \mathbb{C}$，我们需要证明 $\alpha f + \beta g \in L^2([a, b])$。证明：根据复数的模长三角不等式，对任意 $x \in [a, b]$ 有：$$|\alpha f(x) + \beta g(x)| \le |\alpha f(x)| + |\beta g(x)| = |\alpha||f(x)| + |\beta||g(x)|$$利用算术-几何均值不等式（或基本不等式 $(u+v)^2 \le 2u^2 + 2v^2$），将上式两边平方：$$|\alpha f(x) + \beta g(x)|^2 \le \left( |\alpha||f(x)| + |\beta||g(x)| \right)^2 \le 2|\alpha|^2|f(x)|^2 + 2|\beta|^2|g(x)|^2$$对上式在 $[a, b]$ 区间上进行勒贝格积分：$$\int_{a}^{b} |\alpha f(x) + \beta g(x)|^2 dx \le 2|\alpha|^2 \int_{a}^{b} |f(x)|^2 dx + 2|\beta|^2 \int_{a}^{b} |g(x)|^2 dx$$由于 $f, g \in L^2([a, b])$，由定义可知右侧的两个积分 $\int_{a}^{b} |f(x)|^2 dx$ 和 $\int_{a}^{b} |g(x)|^2 dx$ 均小于 $+\infty$。从而左侧的积分也必然有限：$$\int_{a}^{b} |\alpha f(x) + \beta g(x)|^2 dx < +\infty$$结论：线性组合 $\alpha f + \beta g$ 依然属于 $L^2([a, b])$，空间对线性运算封闭，满足向量空间的结构要求。

### 核心部分 II：内积公理的四项验证我们需要验证映射

$\langle \cdot, \cdot \rangle: L^2 \times L^2 \to \mathbb{C}$ 严格满足内积的四个核心公理。

1. 共轭对称性（Conjugate Symmetry）要证：$\langle f, g \rangle = \overline{\langle g, f \rangle}$证明：根据内积定义与复共轭的积分性质：$$\overline{\langle g, f \rangle} = \overline{\int_{a}^{b} g(x)\overline{f(x)} dx} = \int_{a}^{b} \overline{g(x)\overline{f(x)}} dx$$利用复数共轭的代数性质 $\overline{z_1 z_2} = \overline{z_1}\cdot\overline{z_2}$ 以及 $\overline{\overline{z}} = z$：$$\int_{a}^{b} \overline{g(x)} \cdot \overline{\overline{f(x)}} dx = \int_{a}^{b} \overline{g(x)} f(x) dx = \int_{a}^{b} f(x)\overline{g(x)} dx = \langle f, g \rangle$$证毕。
1. 第一变元的线性度（Linearity in the First Argument）要证：$\langle \alpha f_1 + \beta f_2, g \rangle = \alpha\langle f_1, g \rangle + \beta\langle f_2, g \rangle$，其中 $\alpha, \beta \in \mathbb{C}$。证明：依靠勒贝格积分的线性性质，直接展开：$$\begin{aligned}\langle \alpha f_1 + \beta f_2, g \rangle &= \int_{a}^{b} (\alpha f_1(x) + \beta f_2(x))\overline{g(x)} dx \&= \int_{a}^{b} \left( \alpha f_1(x)\overline{g(x)} + \beta f_2(x)\overline{g(x)} \right) dx \&= \alpha \int_{a}^{b} f_1(x)\overline{g(x)} dx + \beta \int_{a}^{b} f_2(x)\overline{g(x)} dx \&= \alpha\langle f_1, g \rangle + \beta\langle f_2, g \rangle\end{aligned}$$证毕。
1. 非负性（Non-negativity）要证：$\langle f, f \rangle \ge 0$证明：$$\langle f, f \rangle = \int_{a}^{b} f(x)\overline{f(x)} dx = \int_{a}^{b} |f(x)|^2 dx$$由于复数模的平方 $|f(x)|^2 \ge 0$ 处处成立，根据积分的保号性（非负函数的积分必非负）：$$\int_{a}^{b} |f(x)|^2 dx \ge 0 \implies \langle f, f \rangle \ge 0$$证毕。
1. 正定性（Positive Definiteness）要证：$\langle f, f \rangle = 0 \iff f = \mathbf{0}$ （此处的 $\mathbf{0}$ 代表零向量，即零函数的等价类 $[0]$）。证明：充分性 ($\Leftarrow$)：若 $f = \mathbf{0}$，意味着代表元 $f(x) = 0 \ \text{a.e.}$，显然其平方积分 $\int_{a}^{b} |f(x)|^2 dx = 0$，故 $\langle f, f \rangle = 0$。必要性 ($\Rightarrow$)：若 $\langle f, f \rangle = 0$，即：$$\int_{a}^{b} |f(x)|^2 dx = 0$$根据测度论核心定理（非负函数积分值为 0，则该函数必几乎处处为 0）：$$|f(x)|^2 = 0 \quad \text{a.e.} \implies f(x) = 0 \quad \text{a.e.}$$根据商空间的定义，所有几乎处处为 0 的函数都属于同一个等价类 $[0]$。而在商空间里，$[0]$ 就是那个唯一的零向量 $\mathbf{0}$。因此，极严格地满足：$$f = \mathbf{0}$$证毕。
