## 量词换序

- $\forall x \forall y A(x,y) \Longleftrightarrow \forall y \forall x A(x,y)$
- $\exists x \exists y A(x,y) \Longleftrightarrow \exists y \exists x A(x,y)$

## 量词否定

- $\neg \forall x A(x) \Longleftrightarrow \exists x \neg A(x)$
- $\neg \exists x A(x) \Longleftrightarrow \forall x \neg A(x)$

## 量词辖域收缩扩张等值式

以下结论都可通过对$B$的真值分类讨论来证明

- $\forall x(A(x)\vee B) \Longleftrightarrow \forall x A(x)\vee B$
- $\forall x(A(x)\wedge B) \Longleftrightarrow \forall x A(x)\wedge B$
- $\forall x(A(x)\to B) \Longleftrightarrow \exists x A(x)\to B$
- $\forall x(B \to A(x)) \Longleftrightarrow B \to \forall x A(x)$
- $\exists x(A(x)\vee B) \Longleftrightarrow \exists x A(x)\vee B$
- $\exists x(A(x)\wedge B) \Longleftrightarrow \exists x A(x)\wedge B$
- $\exists x(A(x)\to B) \Longleftrightarrow \forall x A(x)\to B$
- $\exists x(B \to A(x)) \Longleftrightarrow B \to \exists x A(x)$

## 量词分配等值式

对$\forall x A(x)$的真值分类讨论来证明

- $\forall x(A(x)\wedge B(x)) \Longleftrightarrow \forall x A(x)\wedge \forall x B(x)$
- $\exists x(A(x)\vee B(x)) \Longleftrightarrow \exists x A(x)\vee \exists x B(x)$

## 蕴含否定

- $\neg \forall(A(x)\to B(x)) \Longleftrightarrow \exists x (A(x) \wedge \neg B(x))$
- $\neg \exists(A(x) \wedge B(x)) \Longleftrightarrow \forall x (A(x) \to \neg B(x))$
