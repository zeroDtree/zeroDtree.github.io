## 命题逻辑

- simple statement(简单命题): a declarative sentence(a subject with a predicate) that is either true or false
- connective(联结词): a symbol that connects two statements to form a new statement
- compound statement(复合命题): a statement that is formed by connecting two or more simple statements using connectives
- statement(命题): simple statement or compound statement
- truth values(真值): T(true) or F(false)

| connective        |      symbol       |
| ----------------- | :---------------: |
| Negation(否定)    |      $\neg$       |
| Conjunction(合取) |     $\wedge$      |
| Disjunction(析取) |      $\vee$       |
| Implication(蕴含) |       $\to$       |
| Equivalence(等价) | $\leftrightarrow$ |

| $p$ | $\neg p$ |
| --- | -------- |
| F   | T        |
| T   | F        |

| $p$ | $q$ | $p \wedge q$ |
| --- | --- | ------------ |
| F   | F   | F            |
| F   | T   | F            |
| T   | F   | F            |
| T   | T   | T            |

| --- | --- | ---------- |
| F   | F   | F          |

| T   | T   | T          |

| --- | --- | --------- |
| F   | F   | T         |
| F   | T   | T         |
| T   | F   | F         |
| T   | T   | T         |

| $p$ | $q$ | $p \leftrightarrow q$ |
| --- | --- | --------------------- |
| F   | F   | T                     |

| T   | T   | T                     |

- A statement form(命题形式，命题公式) is an expression involving statement variables and connectives, which can be formed using the rules:
  - (i) Any statement variable is a statement form.
  - (ii) if A and B are statements forms, then $\neg A$, $A \wedge B$, $A \vee B$, $A \to B$, and $A \leftrightarrow B$ are also statement forms.
- tautology(重言式): a statement form that is true for all possible values of its statement variables
- contradiction(矛盾式): a statement form that is false for all possible values of its statement variables
- logically implies(逻辑蕴含): $A$ logically implies $B$ if $A \to B$ is a tautology
- logically equivalent(逻辑等价): $A$ and $B$ are logically equivalent if $A \leftrightarrow B$ is a tautology
- restricted statement forms：在 statement form的基础上限制联结词$\neg$，$\wedge$，$\vee$
- adequate connectives(完备联结词)：An adequate set of connectives is a set such that every truth function canbe represented by a statement form containing only connectives from that set.
- Argument(论证)：An argument is a sequence of statements，$A_1,\cdots,A_n, A$，$A_1,\cdots, A_n$ are premises(前提)，$A$ is conclusion(结论)
- invalid argument(无效论证)：A argument is invalid if it is possible to assign truth values to the statement variables occurring in such a way as to make each of $A_1,\cdots,A_n$ take value T and to make $A$ take value F; Otherwise the argument form is valid.

#### 常用的logically equivalent

- 双重否定：$\neg \neg p \leftrightarrow p$
- 合取交换律：$p \wedge q \leftrightarrow q \wedge p$
- 合取结合律：$(p \wedge q) \wedge r \leftrightarrow p \wedge (q \wedge r)$
- 合取德摩根律：$\neg (p \wedge q) \leftrightarrow \neg p \vee \neg q$
- 合取吸收律：$p \wedge (p \vee q) \leftrightarrow p$
- 析取交换律：$p \vee q \leftrightarrow q \vee p$
- 析取结合律：$(p \vee q) \vee r \leftrightarrow p \vee (q \vee r)$
- 析取德摩根律：$\neg (p \vee q) \leftrightarrow \neg p \wedge \neg q$
- 析取吸收律：$p \vee (p \wedge q) \leftrightarrow p$
- 析取对合取的分配律：$p \vee (q \wedge r) \leftrightarrow (p \vee q) \wedge (p \vee r)$
- 合取对析取的分配律：$p \wedge (q \vee r) \leftrightarrow (p \wedge q) \vee (p \wedge r)$
- 蕴含：$p \to q \leftrightarrow \neg p \vee q$
- 等价：$(p \leftrightarrow q) \leftrightarrow (p \to q) \wedge (q \to p)$
- 逆否：$(p \to q) \leftrightarrow (\neg q \to \neg p)$

#### 性质

1. 若$A$和$A\rightarrow B$都是重言式,则$B$也是重言式
2. 用任意statement form替换重言式A中的命题变量所得到的statement form仍是重言式。
3. 若$C$是$A$中的命题公式，$B$是与$C$等价的命题公式，则用$B$替换$A$中的$C$后得到的命题公式$A'$与$A$逻辑等价。
4. 常见完备联结词：$\{\neg,\wedge\},\{\neg,\vee\},\{\neg,\to\}$
5. 一个论证是有效的当且仅当$((A_1\wedge\cdots\wedge A_n)\to A)$是重言式。
