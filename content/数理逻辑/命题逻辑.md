## 命题逻辑

- **simple statement**(简单命题): a declarative sentence(a subject with a predicate) that is either true or false
- **connective**(联结词): a symbol that connects two statements to form a new statement
- **compound statement**(复合命题): a statement that is formed by connecting two or more simple statements using connectives
- **statement**(命题): simple statement or compound statement
- **truth values**(真值): T(true) or F(false)

| connective        |      symbol       |
| ----------------- | :---------------: |
| Negation(否定)    |      $\neg$       |
| Conjunction(合取) |     $\wedge$      |
| Disjunction(析取) |      $\vee$       |
| Implication(蕴含) |       $\to$       |
| Equivalence(等价) | $\leftrightarrow$ |

| $p$ | $\neg p$ |
| --- | -------- |
| F   | T        |
| T   | F        |

| $p$ | $q$ | $p \wedge q$ |
| --- | --- | ------------ |
| F   | F   | F            |
| F   | T   | F            |
| T   | F   | F            |
| T   | T   | T            |

| $p$ | $q$ | $p \vee q$ |
| --- | --- | ---------- |
| F   | F   | F          |
| F   | T   | T          |
| T   | F   | T          |
| T   | T   | T          |

| $p$ | $q$ | $p \to q$ |
| --- | --- | --------- |
| F   | F   | T         |
| F   | T   | T         |
| T   | F   | F         |
| T   | T   | T         |

| $p$ | $q$ | $p \leftrightarrow q$ |
| --- | --- | --------------------- |
| F   | F   | T                     |
| F   | T   | F                     |
| T   | F   | F                     |
| T   | T   | T                     |

- A **statement form**(命题形式，命题公式) is an expression involving statement variables and connectives, which can be formed using the rules:
  - (i) Any statement variable is a statement form.
  - (ii) if $\mathscr{A}$ and $\mathscr{B}$ are statements forms, then $\neg \mathscr{A}$, $\mathscr{A} \wedge \mathscr{B}$, $\mathscr{A} \vee \mathscr{B}$, $\mathscr{A} \to \mathscr{B}$, and $\mathscr{A} \leftrightarrow \mathscr{B}$ are also statement forms.
- **tautology**(重言式): a statement form that is true for all possible values of its statement variables
- **contradiction**(矛盾式): a statement form that is false for all possible values of its statement variables
- **logically implies**(逻辑蕴含): $\mathscr{A}$ logically implies $\mathscr{B}$ if $\mathscr{A} \to \mathscr{B}$ is a tautology, denoted as $\mathscr{A} \implies \mathscr{B}$
- **logically equivalent**(逻辑等价): $\mathscr{A}$ and $\mathscr{B}$ are logically equivalent if $\mathscr{A} \leftrightarrow \mathscr{B}$ is a tautology, denoted as $\mathscr{A} \iff \mathscr{B}$
- **restricted statement forms**：在 statement form的基础上限制联结词$\neg$，$\wedge$，$\vee$
- **adequate connectives**(完备联结词)：An adequate set of connectives is a set such that every truth function canbe represented by a statement form containing only connectives from that set.
- **Argument**(论证)：An argument is a sequence of statements，$\mathscr{A}_1,\cdots,\mathscr{A}_n,\mathscr{A}$，$\mathscr{A}_1,\cdots,\mathscr{A}_n$ are premises(前提)，$\mathscr{A}$ is conclusion(结论)
- **invalid argument**(无效论证)：A argument is invalid if it is possible to assign truth values to the statement variables occurring in such a way as to make each of $\mathscr{A}_1,\cdots,\mathscr{A}_n$ take value T and to make $\mathscr{A}$ take value F; Otherwise the argument form is **valid**.

#### 性质

1. 若$\mathscr{A}$和$\mathscr{A} \to \mathscr{B}$都是重言式,则$\mathscr{B}$也是重言式
2. 用任意statement form替换重言式$\mathscr{A}$中的命题变量所得到的statement form仍是重言式。
3. 若$\mathscr{C}$是$\mathscr{A}$中的命题公式，$\mathscr{B}$是与$\mathscr{C}$等价的命题公式，则用$\mathscr{B}$替换$\mathscr{A}$中的$\mathscr{C}$后得到的命题公式$\mathscr{A}'$与$\mathscr{A}$逻辑等价。
4. 常见完备联结词：$\{\neg,\wedge\},\{\neg,\vee\},\{\neg,\to\}$
5. 一个论证是有效的当且仅当$((\mathscr{A}_1\wedge\cdots\wedge \mathscr{A}_n)\to \mathscr{A})$是重言式。

#### 常用的logically equivalent

- 双重否定：$\neg \neg p \iff p$
- 合取交换律：$p \wedge q \iff q \wedge p$
- 合取结合律：$(p \wedge q) \wedge r \iff p \wedge (q \wedge r)$
- 合取德摩根律：$\neg (p \wedge q) \iff \neg p \vee \neg q$
- 合取吸收律：$p \wedge (p \vee q) \iff p$
- 析取交换律：$p \vee q \iff q \vee p$
- 析取结合律：$(p \vee q) \vee r \iff p \vee (q \vee r)$
- 析取德摩根律：$\neg (p \vee q) \iff \neg p \wedge \neg q$
- 析取吸收律：$p \vee (p \wedge q) \iff p$
- 析取对合取的分配律：$p \vee (q \wedge r) \iff (p \vee q) \wedge (p \vee r)$
- 合取对析取的分配律：$p \wedge (q \vee r) \iff (p \wedge q) \vee (p \wedge r)$
- 蕴含：$p \to q \iff \neg p \vee q$
- 等价：$(p \leftrightarrow q) \iff (p \to q) \wedge (q \to p)$
- 逆否：$(p \to q) \iff (\neg q \to \neg p)$
